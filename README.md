# Sudoku-Solver-CSP

Programa en Prolog que resuelve Sudokus 9x9 utilizando técnicas de:

Problemas de Satisfacción de Restricciones (CSP)
Propagación de restricciones
Backtracking
Heurísticas de búsqueda
Optimización para resolver grandes volúmenes de Sudokus

El programa está diseñado para procesar archivos de texto con múltiples Sudokus y generar automáticamente un archivo con las soluciones.
